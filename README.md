# laivanupotus
Harjoitustyö pythonin opetteluun

Opiframen Opikoodia kurssilla itse suunniteltu ja ohjelmoitu projektityö. Ohjelma on vielä kesken.

Projektin tarkoituksena oppia Python-ohjelmointia, sekä SqLite3 tietokantojen opettelua.

Aloitin projektin tekemällä perinteisen laivanupotuksen consolille printattavana versiona. Myöhemmin käytin PyGames kirjastoa graafisen ulkoasun tekemiseen.

Onnistuminen projektissa on ollut ihan hyvä. Ongelmaksi on toistaiseksi muodostunut minun lähtökohta ja ajatukset kuinka halusin näyttää laivojen sijainnit ja tapahtumat laudoilla. Graafista ulkoasua tehdessä alussa mietityt asiat tuottavat päänvaivaa.

Projektista puuttuu vielä äänet, oikea alkuvalikko ja monien bugien korjaaminen.
